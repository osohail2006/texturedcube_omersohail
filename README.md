# Interactive Textured Cube (HCI/CG Project)

A 3D interactive application built with **React**, **Three.js**, and **Vite**. This project features a rotatable 3D cube with custom textures representing academic and personal identity.

## 🚀 Features
* Smooth 3D rendering using **WebGL**.
* Custom texture mapping (JPEG/PNG) on 6 distinct faces.
* Responsive design for various screen sizes.

## 🛠️ Tech Stack
* **React.js** - Frontend Framework
* **Three.js** - 3D Library
* **Vite** - Build Tool
